“我将无我，不负人民”，品读习近平今年首访20句妙论

3月21日至26日，国家主席习近平对意大利、摩纳哥、法国进行了国事访问。习近平主席此次欧洲之行是我党和国家最高领导人今年首次出访，对中意、中摩、中法关系发展具有重要历史意义。

习近平主席在6天时间里到访罗马、巴勒莫、摩纳哥、尼斯、巴黎5座城市，密集出席40余场活动，同欧洲领导人共叙友好交往佳话，共谱全面合作新篇，共绘未来发展蓝图。新时代学习工作室梳理习主席本次出访期间的一些精彩论述，供广大网友学习体会。

这么大一个国家，责任非常重、工作非常艰巨。我将无我，不负人民。我愿意做到一个“无我”的状态，为中国的发展奉献自己。一个举重运动员，最开始只能举起50公斤的杠铃，经过训练，最后可以举起250公斤。我相信可以通过我的努力、通过全中国13亿多人民勠力同心来担起这副重担，把国家建设好。我有这份自信，中国人民有这份自信。
